## マップ管理
### general
* スポーンが2つある
* ブロック変更を想定しない
* 10-20個ほど必要
### uhc
* スポーンが2つある
* ブロック変更を想定する
* 立方体の岩盤の中
* 10個ほど必要
* リセット処理が必要
### hole
* [0,0,0]から[31,91,31]のサイズを石で埋める
* 外枠は岩盤で6面を囲う
* スポーン用の空間[8,50,16],[8,51,16],[24,50,16],[24,51,16]
* ブロック変更を想定する
* リセット処理が必要
### underWater
* 32 * 16 * 32が水で満たされたマップ
* スポーン座標は不明
* リセット処理不要
