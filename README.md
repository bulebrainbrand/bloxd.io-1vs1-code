# bloxd.io-1vs1-code
bloxd1vs1のコード
暇人に依頼されてる(githubアカウント未所持)
## 基礎の設計
### 大まかな方針
* oop
* codeloaderは[自作のやつ](https://github.com/bulebrainbrand/bloxdio-codeloader)
* 全体的に追加に強いようにしたい
## マップ管理
### general
* スポーンが2つある
* ブロック変更を想定しない
* 10-20個ほど必要
### uhc
* スポーンが2つある
* ブロック変更を想定する
* 立方体の岩盤の中
* 10個ほど必要
* リセット処理が必要

## ゲームモード
### duel
#### アイテム
* 無エンチャダイヤフル
* ダイヤ剣
#### 詳細
* HP100,シールド0から
* マップはgeneral
* ブロック変更は不可
* 先に殺したほうが勝ち
### uhc
#### アイテム
* エンチャダイヤフル?
* エンチャダイヤ剣?
* ダイヤモンドスパイク(個数不明)
* ネット(個数不明)
* 回復?
#### 詳細
* HP100,シールド0から
* マップはuhc
* ブロック変更可能(マップ範囲内のみ)
* 先に殺したほうが勝ち

### uhcBuild
#### アイテム
* エンチャダイヤフル?
* エンチャダイヤ剣?
* ダイヤモンドスパイク(個数不明)
* ネット(個数不明)
* 回復?
* 足場用ブロック?
#### 詳細
* HP100,シールド0から
* マップはuhc
* ブロック変更可能(マップ範囲内のみ)
* 先に殺したほうが勝ち

#### アイテム
* 無エンチャダイヤフル

* 足場ブロック?
#### 詳細
* HP100,シールド0から
* マップはuhc
